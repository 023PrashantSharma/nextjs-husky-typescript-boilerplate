# Next.js Husky TypeScript Boilerplate

This repository is a boilerplate for creating a Next.js application with TypeScript and Husky pre-configured. It provides a starting point for building scalable and maintainable web applications with best practices.

## Features

- **Next.js**: A React framework for production.
- **TypeScript**: JavaScript with static type checking.
- **Husky**: Git hooks for running scripts before commits.
- **ESLint**: Linting tool for identifying and reporting on patterns in JavaScript.
- **Prettier**: Code formatter to ensure a consistent style.
- **Jest**: JavaScript testing framework.
- **Commitlint**: Linting tool for commit messages.
